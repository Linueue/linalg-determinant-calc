# Determinant Calculator

## How to use

1. Clone from Github

```
git clone https://github.com/Linueue/linalg-determinant-calc.git
```

2. Run it

```
python determinant.py (--test) # optional
```

## Contributions

Check first if there are existing changes,

```
git status
```

If there is, update the local repo using,

```
git pull origin master
```

When you want to upload changes from Github, use,

```
git push origin master
```
